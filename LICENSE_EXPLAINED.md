## 📖 Лицензия Методологии: CC BY-NC-SA 4.0

### Что Это Значит?

**CC BY-NC-SA 4.0** = Creative Commons Attribution-NonCommercial-ShareAlike 4.0

### Разрешено ✅

- **Делиться** — копировать, распространять материал
- **Адаптировать** — модифицировать, улучшать методологию
- **Использовать в исследованиях** — применять в научных работах
- **Цитировать** — использовать в публикациях с указанием авторства
- **Обучать** — использовать в образовательных целях

### Запрещено ❌

- **Коммерческое использование** без соглашения с автором:
  - Продажа торговой стратегии на основе методологии
  - Подписка на торговые сигналы
  - Включение в коммерческий продукт
  - Обучение за плату с использованием методологии

### Условия

1. **Attribution (BY):** Указывайте автора при использовании
2. **NonCommercial (NC):** Не используйте в коммерческих целях без разрешения
3. **ShareAlike (SA):** Делитесь улучшениями на тех же условиях

### Примеры Использования

| ✅ Разрешено | ❌ Запрещено |
| :--- | :--- |
| Исследовательская статья с цитированием | Продажа стратегии "на основе методологии" |
| Образовательный курс (бесплатный) | Платный курс с использованием методологии |
| Личное использование для торговли | Коммерческий продукт с методологией |
| Улучшение методологии с публикацией | Использование без указания авторства |

## 💻 Лицензия Кода: MIT

### Что Это Значит?

**MIT License** — одна из самых свободных лицензий для программного кода.

### Разрешено ✅

- **Коммерческое использование** — использовать код в коммерческих проектах
- **Модификация** — изменять код как угодно
- **Распространение** — распространять исходный и модифицированный код
- **Приватное использование** — использовать в закрытых проектах

### Условия

1. **Включить лицензию** — при распространении кода включите файл LICENSE
2. **Указать авторство** — сохраните копирайт и уведомление о лицензии

### Почему Две Лицензии?

- **Методология (CC BY-NC-SA):** Защищает от коммерциализации без соглашения, но позволяет исследования
- **Код (MIT):** Свободное использование кода, включая коммерческое

## ❓ Часто Задаваемые Вопросы

### Q1: Могу ли я использовать методологию в коммерческом проекте?

**A:** Для коммерческого использования методологии (не кода) требуется соглашение с автором. Свяжитесь через [email для коммерческих запросов] или GitHub Issues.

### Q2: Могу ли я использовать код в коммерческом проекте?

**A:** **Да!** Код под лицензией MIT, коммерческое использование разрешено.

### Q3: Могу ли я продавать стратегию на основе методологии?

**A:** **Нет.** Это считается коммерческим использованием методологии и требует соглашения.

### Q4: Могу ли я использовать методологию в исследовательской статье?

**A:** **Да!** Укажите авторство и ссылку на репозиторий.

### Q5: Могу ли я создать форк и улучшить методологию?

**A:** **Да!** При условии:
- Укажите авторство оригинальной работы
- Используйте ту же лицензию (CC BY-NC-SA 4.0)
- Поделитесь улучшениями (ShareAlike)

### Q6: В чём разница между методологией и кодом?

**A:**
- **Методология** = текст, описание, формулы, концепции (CC BY-NC-SA 4.0)
- **Код** = программная реализация на Python/C++ (MIT)
